# Blog construído em React

Interface para página de blog com suporte a leitura e gerenciamento de postagens, e sistema de comentários.

Esse projeto foi feito construído para uma lição do [The Odin Project](https://theodinproject.com) e funciona
em conjunto com essa [API desenvolvida com express](https://github.com/br-adriel/express-blog-api).
